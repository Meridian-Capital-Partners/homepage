# Meridian Capital Partners — Homepage

Static landing page for [meridiancp.com](https://meridiancp.com).

## Overview

Single-page HTML site for Meridian Capital Partners, a crypto asset recovery firm targeting transnational financial crime with a focus on Southeast Asia.

**Stack:** Vanilla HTML/CSS · Google Fonts (Playfair Display, Inter) · No build step required.

## Structure

```
homepage/
├── index.html   # Single-file landing page
└── README.md
```

## Brand

- **Background:** `#0D0D0D` (near-black)
- **Accent:** `#C41E3A` (crimson)
- **Headlines:** Playfair Display (serif)
- **Body/Labels:** Inter (sans)

## Usage

Open `index.html` directly in a browser — no server required for local preview.

For deployment, serve the file from any static host (Netlify, Vercel, GitHub Pages, etc.).
